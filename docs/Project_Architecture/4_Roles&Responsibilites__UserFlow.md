## Roles and Responsibilities

# 👤 User
Represents individuals registered in the platform.

# 🛠️ Admin
Manages platform content such as banners, categories, and products.

# 📦 Products
Represents all products available in the platform.

# 🛒 Cart
Stores products added by users, identified using User ID.

# 📑 Orders
Stores all orders placed by users in the platform.

---

## User Flow

1. Users register for an account  
2. Users log in with credentials  
3. Users browse available products  
4. Users add products to cart  
5. Users proceed to checkout (address + payment)  
6. Users view orders in profile section
